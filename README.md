# Netflix Clone

Um projeto de clone da Netflix desenvolvido como parte do programa de formação da **Alura**.

## 📋 Descrição

Interface de streaming inspirada na Netflix, construída com **HTML5**, **CSS3** e **JavaScript** vanilla, focando em responsividade e experiência do usuário moderna.

## 🎯 Objetivo

Praticar conceitos fundamentais de desenvolvimento Front-End, criando uma interface web interativa e elegante.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## 📚 Curso

Desenvolvido em conjunto com a **Alura** - Plataforma de educação em tecnologia.

---

*Projeto educacional realizado durante formação em Front-End.*
